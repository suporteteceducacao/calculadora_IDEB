Esta é uma aplicação web para calcular o IDEB preliminar 2023. Basta apenas inserir os valores nas caixas de entrada e o IDEB será calculado quando clicado no botão CALCULAR. É possível também exportar os resultados para um arquivo de impressão bastando para isso clicar no botão PDF.

PROFICIÊNCIA MÉDIA (NOTA SAEB)


*Atenção: utilizar somente ponto (.) para o formato decimal! Ex.: 125.75

TAXA DE RENDIMENTO 2023 POR ETAPA


*Atenção: digitar os valores iguais no formato percentual! Ex.: 98% = 98.
E utilizar somente ponto (.) para os formatos decimais. Ex.: 98.7

*Atenção: Caso a escola NÃO tenha alguma etapa em questão, PREENCHER com ZERO (0) o campo correspondente.
